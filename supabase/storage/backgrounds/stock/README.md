# Stock background images

Drop curated images here, then run `make backgrounds-upload`.

- **Keywords to source (Unsplash):** calm, zen, peaceful, forest
- **Format:** `.jpg` / `.jpeg` / `.png` / `.heic` / `.webp`
- **Naming:** lowercase-kebab, e.g. `zen-01.jpg`, `forest-mist.jpg`
- The app lists everything under `stock/` and shows it in the Stock tab; the
  filename's extension is how it's recognized as an image.

These are served publicly via the `backgrounds` bucket, so only upload images
you have the right to redistribute (Unsplash license is fine — keep attribution
in your records).
